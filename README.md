# libwlmio

A C and Python library for use with the Widgetlords [WL-MIO](https://wlmio.com/) product line.

## Examples

https://wlmio.com/applications.html
https://github.com/widgetlords/wlmio_examples

## Dependencies

- libcanard (MIT)
- libgpiod (LGPL-2.1)

## Package for Raspberry Pi

```
meson build/
make deb
```
